## Around the codebase
# [build](build)
Bourne-Again SHell script to compile the code. Packs all `.c` files into a single `novaos` executable compiled for `x86_64`
# [gpu.c](gpu.c)
Responsible for all functionality related to drawing to the framebuffer.
